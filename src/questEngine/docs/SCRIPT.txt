# SCENE 3 – THE ABYSS
═══════════════════════════════════════════════════════════════════════════════

Location: The Eternal Toroid

Description: A donut-shaped dimension that loops endlessly. Guide Persephone 
            through this liminal space, collecting light while avoiding darkness.

────────────────────────────────────────────────────────────────────────────────
QUEST: PERSEPHONE'S LIGHT
────────────────────────────────────────────────────────────────────────────────

Mechanics:
  → Collect orbs of light
  → Avoid dark shadows
  → Avoid pomegranate-like temptations

UI: Balance Bar
  - Starts centered
  - Darkness collected: bar moves left, turns dark purple
  - Light collected: bar moves right, turns pastel glow
  
Outcomes:
  - Too much darkness → Return to Scene 2, restart quest (click old portal)
  - Enough light → Progress to Scene 4
